Personal Projects


Doma


My main personal project, which I work on infrequently when I have time, is a flashcard website aimed at language learners. I am hoping to deploy this website (and eventually app) as a tool that focuses on strategies that benefit language learning in particular, as opposed to memorization. The website is a work in progress; next steps include:


  • adding user logins
  • adding a database
  • cleaning up website - visual appeal and accessibility
  • implementing security features


In the long term, I also want to implement more games and practice methods, including matching games and some vocab-specific games (eg a paint-by-number for practicing color words, counting games, etc). This is a long-term passion project I hope to work on and eventually deploy as a real resource. I am trying to learn some languages myself, and found myself primarily enjoying resources that have since become paywalled, and I am interested in recreating them for myself and adding to them in ways that actually support language learners based on research describing best practices in language education as well as input from real people.
